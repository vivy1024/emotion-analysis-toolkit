# 开发约定

## 语言

- 项目语言为中文，注释和文档使用中文
- 代码标识符使用英文

## 模块运行方式

各子项目作为 Python 包使用相对导入（`from .utils import ...`），必须以 `-m` 方式运行：

```bash
python -m hidden_emotion_detection.main
python -m micro_expression.train_cnn --config ...
```

## 目录约定

| 目录 | 用途 |
|------|------|
| `hidden_emotion_detection/` | 实时检测系统主代码 |
| `micro_expression/` | 微表情训练代码 |
| `macro_expression/` | 宏表情训练代码 |
| `hidden_emotion_detection/models/` | 模型权重（不纳入 git） |
| `stage1_cnn_output/` | CNN 训练输出（不纳入 git） |
| `stage2_lstm_output/` | LSTM 训练输出（不纳入 git） |
| `docs/` | 项目文档 |

## 配置管理

- 实时检测系统配置：`hidden_emotion_detection/config/config.json`
- 训练配置：各子项目下的 YAML 文件
- 模型路径使用绝对路径，部署到新环境时需修改 `config.json`

## Git 规范

- 模型权重（`.pth`、`.pt`）不提交到 git
- 训练输出（图片、日志）不提交到 git
- `__pycache__/` 不提交到 git
