# 环境搭建

## 系统要求

- Python 3.7+
- CUDA（可选，GPU加速推理和训练）

## 安装依赖

各子项目有独立的 `requirements.txt`：

```bash
# 实时检测系统
pip install -r hidden_emotion_detection/requirements.txt

# 微表情训练
pip install -r micro_expression/requirements.txt

# 宏表情训练
pip install -r macro_expression/requirements.txt
```

核心依赖：PyTorch、OpenCV、PyQt5、dlib、MediaPipe、scikit-learn。

## 模型文件

模型权重存放在 `hidden_emotion_detection/models/`，不纳入 git 版本控制。

主要模型文件：
- `macro.pt` — 宏表情识别模型
- `casme2_cnn_stage1_balanced_cnn_best.pth` — 微表情 CNN 特征提取器
- `lstm_ensemble_models/` — 微表情 LSTM 集成模型
- `AU_predictors/` — AU 检测 SVM 预测器
- `shape_predictor_68_face_landmarks.dat` — dlib 68点人脸关键点
- `svm_models/` — AU 分类 SVM 模型
